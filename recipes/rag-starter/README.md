# RAG Starter: In-Memory TF-IDF Search

Build a minimal retrieval-augmented generation pipeline using TF-IDF for document retrieval.

## What You'll Learn

- How TF-IDF scoring works for document retrieval
- How to evaluate retrieval quality with hit rate metrics
- Basics of building a search index

## Prerequisites

- Node.js >= 20

## Quick Start

```bash
trend2trial recipe init rag-starter ./my-rag
trend2trial recipe run ./my-rag
cat ./my-rag/REPORT.md
```
