# LLM Observability: Structured Tracing

Build a mock LLM call chain with structured JSON tracing and analyze trace completeness.

## What You'll Learn

- How to instrument LLM call chains with structured traces
- How to measure span coverage and detect gaps
- Basics of LLM observability patterns

## Prerequisites

- Node.js >= 20

## Quick Start

```bash
trend2trial recipe init llm-observability-starter ./my-obs
trend2trial recipe run ./my-obs
cat ./my-obs/REPORT.md
```
